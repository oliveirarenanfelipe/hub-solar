# Story 0.1 — Discovery: Mapear Distribuidores de Energia Solar

## Status
Draft

## Contexto
Antes de construir qualquer integração, precisamos entender quais distribuidores existem no mercado brasileiro, quais possuem API, quais precisam de scraping e quais só disponibilizam preços via planilha/PDF/WhatsApp.

## Objetivo
Criar um mapa completo dos distribuidores de energia solar no Brasil com o tipo de acesso aos preços de cada um, para guiar a arquitetura do HUB Solar.

## Acceptance Criteria

- [ ] Listar no mínimo 10 distribuidores relevantes no Brasil
- [ ] Para cada distribuidor: nome, site, região de atuação, tipo de acesso aos preços (API / Site scraping / PDF / WhatsApp / Outro)
- [ ] Identificar quais distribuidores possuem API pública ou catálogo online
- [ ] Documentar em `docs/research/distribuidores.md`
- [ ] Definir os 3 distribuidores prioritários para integração na Fase 1

## Tasks

- [ ] Pesquisar principais distribuidores (Google, LinkedIn, feiras do setor)
- [ ] Visitar sites de cada distribuidor para identificar catálogos/preços online
- [ ] Testar se algum possui API pública ou login com acesso a preços
- [ ] Criar `docs/research/distribuidores.md` com mapeamento completo
- [ ] Priorizar 3 distribuidores para Fase 1 e justificar escolha
