*Data: 2026-03-19*

## SolarApp (https://solarapp.com.br)

**Propósito:** SaaS de dimensionamento e gestão para integradores solares. Gera propostas em < 1 minuto.

**Funcionalidades:**
- Dimensionamento com Lei 14.300
- Gerador de propostas com kits automáticos
- Parametrização de margens por projeto
- Integração com distribuidores (cadastro manual de kits)
- Histórico de compras, gestão de pós-venda

**Acesso a preços:** Cadastro manual pelo integrador — sem integração automática com APIs de distribuidores.

**Modelo:** Freemium. Monetiza via comissão em vendas realizadas pela plataforma.

**Gap:** Sem integração automática de preços. Dependente de cadastro manual.

---

## Suns Brasil (https://lp.sunsbrasil.com.br)

**Propósito:** Ecossistema completo para integradores — marketplace + CRM + proposta + financiamentos.

**Funcionalidades:**
- Marketplace multi-fornecedor com cotação comparativa
- Gerador de propostas gratuito
- CRM integrado
- Geração de leads
- Financiamentos (parcerias com financeiras)
- Homologação técnica em todo Brasil
- Seguros especializados
- Cashback em materiais
- Comunidade com 600+ integradoras

**Acesso a preços:** Aparentemente integração direta com distribuidoras (não apenas manual).

**Modelo:** Freemium + marketplace com comissão + spread em financiamentos.

**Gap:** Muito focado em PMEs. Pouca ferramenta para autônomos individuais. Analytics limitado.

## Insights Estratégicos

### Nossa diferenciação potencial

1. **Integração automática de preços** (não manual) — inclusive para distribuidores sem API via scraping
2. **Comparação lado a lado** de distribuidores para maximizar margem
3. **Foco no autônomo** (não em PME) — interface simples, sem bloat de CRM/seguros
4. **Transparência de preços** — mostrar de onde veio o preço e quando foi atualizado

### Modelo de negócio sugerido

- Fase 1: Uso próprio (interno) — sem cobrança
- Fase 2: Saas para outros integradores autônomos
- Monetização: Assinatura mensal (não comissão por transação)
